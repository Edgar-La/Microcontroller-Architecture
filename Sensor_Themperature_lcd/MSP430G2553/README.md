# Sensor de temperatura

Aqui se encuentra el código del [programa](https://github.com/Edgar-La/Microcontroller-Architecture/blob/main/Sensor_Themperature_lcd/MSP430G2553/main.c) que muestra el uso del __sensor de temperatura LM35__ el cual es un sensor comercial.

Para esto se ha programado el uso de un __convertidor analógico digital con referencia externa__.

Además se ha usado una pantalla LCD 16x2 para desplegar el resultado.

__* Nota:__ Las librerias de la pantalla LCD son propiedad de _Elliott Gurrola_, pero están disponibles en la web.
